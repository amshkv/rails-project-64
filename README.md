[Daily Prophet](https://daily-prophet.mshkv.ru/) is a simple blog application. It allows you to create posts, add comments, and subscribe to other users.

## Requirements

- Ruby 3.1.2
- NodeJS 16 and later
- Yarn
- Gitmoji CLI

## Setup

```sh
make setup
```

## Run

```sh
make start
```

## Development

### Install hooks

```sh
make install-hooks
```

### Tests

```sh
make test
```

### Linter

```sh
make lint
```
